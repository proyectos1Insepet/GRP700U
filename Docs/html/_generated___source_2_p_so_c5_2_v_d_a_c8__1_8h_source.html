197 /*******************************************************************************
198 * Trim
199 * Note - VDAC trim values are stored in the "Customer Table" area in * Row 1 of
200 *the Hidden Flash. There are 8 bytes of trim data for each VDAC block.
201 * The values are:
202 * I Gain offset, min range, Sourcing
203 * I Gain offset, min range, Sinking
204 * I Gain offset, med range, Sourcing
205 * I Gain offset, med range, Sinking
206 * I Gain offset, max range, Sourcing
207 * I Gain offset, max range, Sinking
208 * V Gain offset, 1V range
209 * V Gain offset, 4V range
210 *
211 * The data set for the 4 VDACs are arranged using a left side/right side
212 * approach:
213 * Left 0, Left 1, Right 0, Right 1.
214 * When mapped to the VDAC0 thru VDAC3 as:
215 * VDAC 0, VDAC 2, VDAC 1, VDAC 3
216 *******************************************************************************/
217 #define VDAC8_1_TRIM_M7_1V_RNG_OFFSET 0x06u
218 #define VDAC8_1_TRIM_M8_4V_RNG_OFFSET 0x07u
